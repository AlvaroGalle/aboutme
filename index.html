Sobre Mí

¡Hola! Soy Álvaro Gallego, un apasionado desarrollador web con experiencia en la creación de aplicaciones innovadoras y funcionales. Me especializo en el desarrollo backend utilizando Java y Spring Boot, así como en el desarrollo frontend con Angular y Tailwind CSS. Mi objetivo es crear soluciones tecnológicas que no solo sean eficientes, sino también atractivas y fáciles de usar.

A lo largo de mi carrera, he trabajado en diversos proyectos que me han permitido adquirir habilidades sólidas en el diseño y la implementación de aplicaciones web. Me encanta enfrentar desafíos técnicos y encontrar soluciones creativas para satisfacer las necesidades de los usuarios.

Estoy constantemente aprendiendo y actualizándome con las últimas tendencias en desarrollo web para ofrecer soluciones modernas y efectivas. Si estás buscando a alguien apasionado por la tecnología y comprometido con la excelencia, ¡no dudes en contactarme!

Proyectos

Gymtracker

El proyecto utiliza Java con Spring Boot para desarrollar el backend y gestionar la lógica de la aplicación. El frontend está desarrollado con Thymeleaf e incorpora Bootstrap para ofrecer una interfaz web intuitiva y responsiva. La aplicación utiliza MySQL como base de datos principal y H2 como base de datos embebida para la API. Todo el sistema se despliega de forma sencilla y portátil mediante Docker, lo que facilita su instalación y ejecución en diferentes entornos.

PlanAula

PlanAula es una aplicación web creada en Java que emplea PostgreSQL como base de datos para almacenar y administrar de forma eficiente la información relacionada con horarios escolares, como aulas, asignaturas, profesores y tramos horarios. Su propósito es optimizar la organización académica mediante una gestión clara y visual de la planificación semanal. El proyecto está configurado para su despliegue en entornos Docker y Render, lo que facilita su portabilidad y mantenimiento.
