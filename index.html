Proyectos

Gymtracker

El proyecto utiliza Java con Spring Boot para desarrollar el backend y gestionar la lógica de la aplicación. El frontend está desarrollado con Thymeleaf e incorpora Bootstrap para ofrecer una interfaz web intuitiva y responsiva. La aplicación utiliza MySQL como base de datos principal y H2 como base de datos embebida para la API. Todo el sistema se despliega de forma sencilla y portátil mediante Docker, lo que facilita su instalación y ejecución en diferentes entornos.

PlanAula

PlanAula es una aplicación web creada en Java que emplea PostgreSQL como base de datos para almacenar y administrar de forma eficiente la información relacionada con horarios escolares, como aulas, asignaturas, profesores y tramos horarios. Su propósito es optimizar la organización académica mediante una gestión clara y visual de la planificación semanal. El proyecto está configurado para su despliegue en entornos Docker y Render, lo que facilita su portabilidad y mantenimiento.
